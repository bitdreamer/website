The Bit Lab
See links below for more pictures.

The Bit Lab is student built and run computer lab.   The computer students have built high end PCs, installed Linux operating systems and networking.  The lab is a playground for computer students run web servers, stereo projection graphics, advanced programming and anything else we can think of.

In the beginning (1999), the Bit Lab was just an idea.  We had a  proposal, a vision for what it could be.  It's gratifying to see how much of our current reality is a fulfillment of this dream.
