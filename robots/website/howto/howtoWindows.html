Our first piece of advice is to read the README file. You will find that it can be very informative and helpful, and it will probably tell you some things that we won't cover here.
Our second piece of advice is to pick a topic from the list below for step by step instructions that even a 10 yr old could follow.

  • Install leJOS

  • Load firmware to RCX

  • Load programs to RCX



How to install leJOS

1) Download the newest version of Java (JDK) from the Sun site

2) Unzip the JDK file using Winzip or some other application. Typically, all Windows OS come equipped with a version of Winzip. If you don't have Winzip or another application, go to the Winzip site and download a copy of it. It's very easy to use.

3) Download the correct version of leJOS from the leJOS site

4) Unzip the leJOS file using Winzip or some other application.

5) Pull up the Start Menu, go to Programs then Accessories and open a DOS command prompt. If it's not there, pull up the Start Menu, click "Run.." and type "command" then click "OK" and a DOS command prompt should appear.

6) Add the JDK bin and the leJOS bin folder to your PATH. Type "PATH" then hit the enter key to view your current PATH. Type "set PATH=(insert your current PATH);(insert full address for leJOS)\bin;(insert full address for JDK)\bin" followed by the enter key. For example, I would type "set PATH=C:\WINDOWS\system32;C:\WINDOWS;C:\WINDOWS\System32\Wbem;C:\PROGRAM FILES\THINKPAD\UTILITIES;C:\WINDOWS\System32;C:\lejos\bin;C:\j2sdk1.4.0_01\bin" because C:\WINDOWS\system32;C:\WINDOWS;C:\WINDOWS\System32\Wbem;C:\PROGRAM FILES\THINKPAD\UTILITIES;C:\WINDOWS\System32 is my current PATH and my copies of leJOS and JDK aren't in any other folders, just on the C drive.

7) set the RCXTTY variable to the infared tower. For example, if your IR tower is USB, it would be "set RCXTTY=USB". If your IR tower is not USB, it might be something like this "set RCXTTY=COM1" or "set RCXTTY=COM2".

NOTE: steps 6 and 7 must be done each time a new DOS prompt is opened.

How to load the firmware to the RCX

1) Make sure the PATH and RCXTTY are correct by typing "set" into a DOS prompt and check the list for those variables and their values. (see steps 6 and 7 of How to install leJOS to Windows)

2) Turn the RCX on and position the RCX in front of the IR tower.

3) Go to the leJOS bin folder by typing "cd (insert full address of leJOS)\bin" and then type "lejosfirmdl -f". When the firmware starts, the LCD (screen on the RCX) shows the battery voltage, and you should hear a double beep.

NOTE: Downloading the firmware only needs to be done once and it can easily be overwritten by the LEGO firmware that comes on the CD-ROM.

How to load programs to the RCX

1) Make sure the PATH and RCXTTY are correct by typing "set" into a DOS prompt and check the list for those variables and their values. (see steps 6 and 7 of How to install leJOS to Windows)

2) Turn the RCX on and position the RCX in front of the IR tower.

3) Go to the folder where your program is by typing "cd (insert full address of that folder)"

4) Type "set CLASSPATH=.".

5) Compile each .java file in the folder by typing "lejosc *.java". Or compile each .java file individually by typing "lejosc (filename).java".

6) Once successfully compiled, load the program to the RCX by typing "lejos (filename of the file that has the main method)". Note the absence of the ".java".
