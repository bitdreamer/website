Our first piece of advice is to read the README file. You will find that it can be very informative and helpful, and it will probably tell you some things that we won't cover here.
Our second piece of advice is to pick a topic from the list below for step by step instructions that even a 10 yr old could follow.

  • Install leJOS

  • Load firmware to RCX

  • Load programs to RCX



How to install leJOS

1) Download the correct version of leJOS from the leJOS site

2) Go to the directory you downloaded leJOS to and unzip the leJOS file using gunzip or some other application. Type "gunzip (insert file name)".

3) Go to the directory you downloaded leJOS to and untar the leJOS file using the tar command. Type "tar xvf (insert file name)".

4) Add the JDK bin and the leJOS bin folder to your PATH. Type "export PATH=$PATH:(insert full address for leJOS)\bin:(insert full address for JDK)\bin" followed by the enter key. For example, I would type "export PATH=$PATH:/usr/local/lejos/bin:/usr/local/j2sdk1.4.0_01/bin" because my copies of leJOS and JDK are located in the /usr/local directory.

5) Go to the leJOS directory and run the Makefile by simply typing "make". Please notice it's lowercase. This step creates the tools and scripts necessary to run leJOS.

6) Set the RCXTTY variable to the infared tower. For example, if you are using the serial IR tower, it might be something like this "export RCXTTY=/dev/ttyS1" or "export RCXTTY=/dev/ttyS2" where the numbers 1 and 2 represent the serial port number.

NOTE: steps 4 and 6 must be done each time a new window is opened. As an alternative, you could easily create a shell script to run each time and save you a step or two.

How to load the firmware to RCX

1) Make sure the PATH and RCXTTY are correct by typing "echo $PATH" and "echo RCXTTY" into a window. (see steps 4 and 6 of How to install leJOS)

2) Turn the RCX on and position the RCX in front of the IR tower.

3) Type "lejosfirmdl -f". When the firmware starts, the LCD (screen on the RCX) shows the battery voltage, and you should hear a double beep.

NOTE: Downloading the firmware only needs to be done once and it can easily be overwritten by the LEGO firmware that comes on the CD-ROM.

How to load programs to RCX

1) Make sure the PATH and RCXTTY are correct by typing "echo $PATH" and "echo RCXTTY" into a DOS prompt. (see steps 4 and 6 of How to install leJOS)

2) Turn the RCX on and position the RCX in front of the IR tower.

3) Go to the folder where your program is by typing "cd (insert full address of that folder)"

4) Type "export CLASSPATH=.".

5) Compile each .java file in the folder by typing "lejosc *.java". Or compile each .java file individually by typing "lejosc (filename).java".

6) Once successfully compiled, load the program to the RCX by typing "lejos (filename of the file that has the main method)". Note the absence of the ".java".
